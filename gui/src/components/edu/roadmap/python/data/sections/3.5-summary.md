# 3.5 요약

## 📌 3장 요약
✅ 리스트(List): 순서가 있는 데이터 저장, 수정 가능  
✅ 튜플(Tuple): 순서가 있지만 수정 불가능  
✅ 딕셔너리(Dictionary): 키-값 쌍으로 저장하며 키를 이용해 데이터 검색  
✅ 집합(Set): 중복 없는 데이터 저장, 순서 없음  
✅ 리스트 컴프리헨션을 사용하면 간결한 코드 작성 가능
✅ 다양한 데이터 구조를 상황에 맞게 조합하여 사용하면 효율적인 코드 작성이 가능합니다
