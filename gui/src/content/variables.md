# 변수와 자료형
## 변수 선언과 기본 자료형

## 변수
```python
x = 10
name = "Python"
is_active = True
```

### 명명 규칙
- 문자, 숫자, 밑줄(_) 조합
- 숫자 시작 불가
- 대소문자 구분
- 예약어 사용 불가

## 기본 자료형
```python
# 정수
a = 42

# 실수
pi = 3.14

# 문자열
s = 'Hello'

# 불리언
t = True

# None
n = None
```
