# 집합
## 중복되지 않는 요소 모음

## 집합 생성
```python
unique_numbers = {1, 2, 2, 3}  # {1, 2, 3}
empty_set = set()
```

## 집합 연산
```python
A = {1, 2, 3}
B = {3, 4, 5}

print(A | B)  # 합집합: {1, 2, 3, 4, 5}
print(A & B)  # 교집합: {3}
```
