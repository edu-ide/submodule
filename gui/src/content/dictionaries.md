# 딕셔너리
## 키-값 쌍 컬렉션

## 딕셔너리 생성
```python
student = {
    'name': 'Alice',
    'age': 20,
    'major': 'Computer Science'
}
```

## 딕셔너리 조작
```python
# 값 접근
print(student['name'])  # Alice

# 항목 추가
student['gpa'] = 3.8
```
