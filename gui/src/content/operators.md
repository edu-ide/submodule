# 연산자
## 산술, 비교, 논리 연산자

## 산술 연산자
```python
print(10 + 3)   # 13
print(10 - 3)   # 7
print(10 * 3)   # 30
print(10 ** 3)  # 1000
```

## 비교 연산자
```python
print(5 > 3)    # True
print(5 == 3)   # False
```

## 논리 연산자
```python
print(True and False)  # False
print(True or False)   # True
```
