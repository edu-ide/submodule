## 주요 자료구조
- **리스트(List)**: 순서 O, 변경 가능
- **튜플(Tuple)**: 순서 O, 변경 불가
- **세트(Set)**: 순서 X, 중복 불가
- **딕셔너리(Dictionary)**: 키-값 쌍

```python
# 리스트 예제
numbers = [1, 2, 3]

# 튜플 예제
coordinates = (37.5, 127.0)

# 세트 예제
unique_numbers = {1, 2, 2, 3}  # {1, 2, 3}

# 딕셔너리 예제
person = {'name': 'Alice', 'age': 30}
```
