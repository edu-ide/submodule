# 조건문
## if, else, elif 구문

## 기본 구조
```python
if condition:
    # 코드 블록
elif another_condition:
    # 코드 블록
else:
    # 코드 블록
```

## 예제: 성인 여부 확인
```python
age = 20
if age >= 19:
    print("성인")
else:
    print("미성년자")
```
