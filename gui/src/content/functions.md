# 함수
## 코드 재사용을 위한 블록

## 함수 정의
```python
def greet(name):
    return f"Hello, {name}!"
```

## 매개변수 종류
```python
def profile(name, age=30):  # 기본값 매개변수
    print(f"{name}: {age}")
```
